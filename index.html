Games collection

A collection of some of my thoughout my time at uni including some personal projects for hoing my skills

Burglar Battle (Unity)

This was a large sale assignment that involved all 40+ memebers of the course. Burglar Battle is 4 player couch co-op game where 2 opposing teams have limited time to make thier way to a vault, while dodging gaurds & traps, and solving puzzles along the way. My major role was desinging the second floor of the game, along with some traps some set dressing.

Tools: Unity 2022.1.16, Visual Studio 2019

CGD Burglar Battle Repo

Precarious Pillars (Tricky Towers De-make)

First of three 3rd year group assignments (7 memebers). We were tasked with de-making a contemporary game to run on the Atari, our choice was tricky towers, that was reduced to 2 player only with 1 game mode; the first to build a tower wins. I worked on the main game loop; setting the main menu, game scene and end game scene

Tools: Unity 2022.1.16, Visual Studio 2019

Technicolour Dream Cavern

Second of three 3rd year group assignments (7 memebers). This task involved making a vertical slice of an original game that could run on limited hardware (single core, 4GB RAM) at 60fps and 1080p. I took part in the level design, gameloop along with 2 others

Tools: Unity 2022.1.16, Visual Studio 2019

Nuts and Beans: Mindy Anna jones and the Technicolour Dream Cavern Repo

Global Game Jam 2023:
Route to the root (of all evil)

Final 3rd year group assignment (7 memebers). This task was taking part in the 2023 Global Game Jam, with the Theme 'Roots'. In our game the player has a limited time to send adventures into an uncharted dungeon to find and kill a boss all while trying to survive the monsters, obsticles and traps hidden within. I was tasked with designing the dungeon layout, obsticle tiles, the UI and the main menu/end game scenes and assisted with implamenting the obsticle behaviour and game loop

Tools: Unity 2022.1.16, Visual Studio 2019

The Idol

2nd year project exploring level design theory, with this verticle slice using the enviromental layout, and lighting to guide the player and hint at what their next move should be. After stealing an ancient artifact the player must traverse a mystical temple full of traps and puzzles, using their special powers, so they can return it to it's rightful place'
